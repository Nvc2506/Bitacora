Investigación de TIC

ANÁLISIS Y DESARROLLO DE SOFTWARE

1. Algoritmo

Un algoritmo es un conjunto finito, ordenado y definido de pasos, instrucciones o reglas lógicas que se utilizan para resolver un problema, realizar un cálculo o llevar a cabo una tarea determinada.

2. Aplicación

La aplicación de los algoritmos es esencial en numerosos campos. En la informática, por ejemplo, son el corazón de todos los programas, aplicaciones y sistemas.

3. Base de datos

Colección organizada de datos que permite almacenar, consultar y actualizar información de manera eficiente.

4. Binario

Sistema de numeración que utiliza solo dos dígitos: 0 y 1.

5. Bitácora

Registro ordenado y detallado de actividades, hechos o procesos realizados en un período de tiempo.

7. Bug

Error o fallo en un programa que causa un comportamiento inesperado o incorrecto.

8. Cloud (Nube)

Modelo tecnológico que permite almacenar, gestionar, procesar y acceder a datos, programas y servicios a través de Internet.

9. Código

Conjunto de instrucciones escritas en un lenguaje de programación para que un computador realice tareas específicas.

10. Compilador

Programa que traduce el código escrito en un lenguaje de alto nivel a un lenguaje máquina ejecutable.

11. CPU (Unidad Central de Procesamiento)

El cerebro del computador que ejecuta instrucciones y procesa datos.

12. Data (Datos)

Conjunto de hechos, cifras o información que pueden ser procesados por sistemas informáticos.

13. Decimal (Número decimal)

Sistema de numeración que utiliza diez dígitos del 0 al 9.

14. Debugging

Proceso de identificar, analizar y corregir errores en un programa o aplicación.

15. Firewall

Sistema de seguridad informática que controla el tráfico de datos entre una red interna y externa.

16. Hardware

Parte física de un sistema informático, como la CPU, memoria, discos duros, monitores, teclado y mouse.

17. HTML

Lenguaje de marcado utilizado para estructurar contenido en páginas web.

18. HTTP/HTTPS

Protocolos de comunicación que permiten la transferencia de información entre servidores y navegadores web.

19. Inteligencia Artificial (IA)

Rama de la informática que desarrolla sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.

20. Tecnologías de la Información y la Comunicación (TIC)

Conjunto de herramientas, recursos y sistemas que permiten crear, procesar, almacenar, transmitir y compartir información digital.

21. Kernel

Parte central de un sistema operativo que gestiona recursos, memoria, procesos y comunicación entre hardware y software.

22. Malware

Software malicioso que causa daño, roba información o interfiere con el funcionamiento de un sistema.

23. Networking (Redes)

Conjunto de dispositivos y protocolos que permiten la comunicación entre computadores.

24. Open Source

Software cuyo código es abierto y accesible para que cualquiera pueda usarlo, modificarlo y distribuirlo.

25. Python

Lenguaje de programación de alto nivel, sencillo y versátil, usado para desarrollo web, análisis de datos e inteligencia artificial.

26. Router

Dispositivo de red que dirige el tráfico de datos entre redes y permite que múltiples dispositivos se conecten a internet.

27. Red neuronal

Es un sistema computacional inspirado en el cerebro humano, compuesto por capas de nodos (neuronas) interconectados que procesan información para aprender de los datos y realizar tareas complejas como reconocer patrones, clasificar objetos o predecir resultados.

28. Seguridad informática

Conjunto de prácticas y herramientas que protegen sistemas, datos y redes frente a ataques, accesos no autorizados o pérdida de información.

29. Servidor

Computador o programa que proporciona servicios y recursos a otros dispositivos o usuarios en una red.

30. Sistema operativo

El software principal que controla y administra todos los recursos de una computadora, dispositivo móvil u otro equipo electrónico. Actúa como intermediario entre el hardware y los programas, permitiendo que el dispositivo funcione correctamente.

31. Software

Conjunto de instrucciones y programas que permiten al hardware realizar tareas específicas. Incluye aplicaciones, sistemas operativos y programas de desarrollo.

32. Storage (Almacenamiento)

Espacio donde se guardan datos digitales. Puede ser interno (disco duro, SSD) o externo (memorias USB, discos externos o nube).

33. Streaming

Transmisión de contenido multimedia en tiempo real a través de internet, como videos, música o transmisiones en vivo.

34. Switch

Dispositivo de red que conecta varios computadores dentro de una misma red local, facilitando la comunicación eficiente entre ellos.

35. Tablet

Dispositivo portátil con pantalla táctil, capaz de ejecutar aplicaciones y conectarse a internet, combinando funciones de computador y smartphone.

36. Terminal

Interfaz que permite al usuario comunicarse directamente con el sistema operativo usando comandos de texto.

37. Token

Elemento de seguridad digital que sirve para autenticar la identidad de un usuario en sistemas y aplicaciones.

38. URL (Localizador Uniforme de Recursos)

Dirección única que identifica y localiza recursos en internet, como páginas web, imágenes o documentos.

39. Usuario

Persona que interactúa con un sistema informático, aplicación o red, usando credenciales y permisos específicos.

40. Virtualización

Tecnología que permite crear versiones virtuales de recursos físicos, como servidores, sistemas operativos o redes, optimizando su uso y reduciendo costos.

41. Virus

Programa malicioso diseñado para replicarse y propagarse en sistemas informáticos, causando daños o robando información.

42. VPN

Una VPN es una tecnología que permite crear una conexión segura y privada a través de Internet, actuando como un “túnel” que protege la comunicación entre tu dispositivo y la red a la que te conectas. Gracias a este sistema, los datos que envías o recibes viajan cifrados (encriptados), tu dirección IP se oculta y tu ubicación real permanece protegida, brindándote un alto nivel de seguridad, anonimato y libertad digital.

43. Web

La web, también conocida como World Wide Web (WWW), es un sistema global que permite acceder, compartir y visualizar información a través de Internet mediante páginas interconectadas por enlaces (hipervínculos). Funciona como un enorme conjunto de documentos digitales —textos, imágenes, videos, audios y otros recursos multimedia— que se pueden consultar desde cualquier parte del mundo con un navegador web.

44. Wireless (Inalámbrico)

Tecnología que permite la transmisión de datos sin cables, utilizando señales de radio, Wi-Fi, Bluetooth o infrarrojo.

45. XML

Lenguaje de marcado utilizado para almacenar y transportar datos de manera estructurada, facilitando el intercambio entre sistemas distintos.

46. API (Interfaz de Programación de Aplicaciones)

Una API (Application Programming Interface o Interfaz de Programación de Aplicaciones) es un conjunto de reglas, protocolos y herramientas que permite que diferentes programas, aplicaciones o sistemas informáticos se comuniquen e interactúen entre sí de manera estructurada y segura. Su principal función es facilitar la integración entre sistemas sin que los desarrolladores tengan que preocuparse por los detalles internos de cada aplicación.

47. Backup (Copia de seguridad)

Proceso de duplicar y almacenar datos importantes para protegerlos frente a pérdida, daños o fallos del sistema.

48. Big Data

Conjunto de técnicas y herramientas que permiten almacenar, procesar y analizar grandes volúmenes de datos para obtener información útil.

49. Bit

Unidad mínima de información en informática, representada como 0 o 1. Múltiples bits forman bytes, que representan datos más complejos.

50. Blockchain

Tecnología de registro distribuido que garantiza transparencia y seguridad de transacciones digitales, utilizada en criptomonedas y contratos inteligentes.

51. Bot

Programa automatizado que ejecuta tareas repetitivas o responde a usuarios, como chatbots en atención al cliente.

52. Cloud Computing

Modelo de servicios que permite almacenar y procesar datos en servidores remotos, accesibles desde cualquier dispositivo con internet.

53. Código fuente

Conjunto de instrucciones escritas por un programador en un lenguaje de programación, que luego se compilan o interpretan para ser ejecutadas por un computador.

54. Conectividad

Capacidad de los dispositivos de comunicarse entre sí mediante redes, cables o tecnologías inalámbricas.

55. Cronograma

Plan de actividades con fechas y tiempos establecidos para organizar tareas, comúnmente usado en proyectos de desarrollo de software.

56. Cyberseguridad

Disciplina que protege la información, redes y sistemas frente a accesos no autorizados, ciberataques y amenazas digitales.

57. Dashboard

Panel visual que muestra información y métricas relevantes de un sistema o aplicación, facilitando el análisis de datos.

58. Debug

Proceso de encontrar y corregir errores en un programa para que funcione correctamente.

59. Desarrollo web

Proceso de creación y mantenimiento de sitios y aplicaciones web, que combina diseño, programación y bases de datos.

60. Desarrollo móvil

Creación de aplicaciones para dispositivos móviles, usando lenguajes como Python, JavaScript, Kotlin o Swift.

61. Ethernet

Ethernet es una tecnología de red de área local (LAN) que permite la comunicación y el intercambio de datos entre computadoras, servidores, impresoras u otros dispositivos conectados mediante cables (generalmente de par trenzado o fibra óptica).

62. Firewall personal

Programa o dispositivo que protege un computador individual de amenazas externas, bloqueando conexiones sospechosas.

63. Framework

Conjunto de herramientas, librerías y convenciones que facilitan el desarrollo de software, acelerando la creación de aplicaciones.

64. FTP (Protocolo de Transferencia de Archivos)

El FTP (File Transfer Protocol o Protocolo de Transferencia de Archivos) es un protocolo de comunicación de red que funciona sobre TCP/IP y cuya finalidad principal es permitir la transferencia de archivos entre un cliente y un servidor, ya sea en una red local o a través de Internet. Se trata de una de las tecnologías más antiguas y ampliamente utilizadas para el intercambio de datos digitales, ya que fue desarrollado en la década de 1970 y estandarizado en el RFC 959.

65. Hosting

Servicio que permite almacenar y publicar sitios web en servidores conectados a internet, garantizando su disponibilidad para los usuarios.

66. HTML (HyperText Markup Language)

Lenguaje de marcado estándar utilizado para estructurar el contenido de páginas web mediante etiquetas.

67. HTTP (Protocolo de Transferencia de Hipertexto)

Protocolo de comunicación que permite la transmisión de información en la web entre clientes y servidores.

68. HTTPS (HTTP Secure)

Versión segura del protocolo HTTP que utiliza cifrado (SSL/TLS) para proteger la transferencia de datos en la web.

69. GraphQL

es una solicitud escrita en el lenguaje GraphQL que un cliente (como una aplicación web o móvil) envía a un servidor para obtener datos específicos, solo aquellos que necesita y nada más, evitando el sobreenvío de información y las múltiples llamadas que a veces se requieren con APIs RESTful.

70. IoT (Internet de las Cosas)

Red de objetos físicos interconectados a través de internet, que pueden recopilar, enviar y recibir datos.

71. IP (Protocolo de Internet)

Identificador único asignado a cada dispositivo conectado a una red para permitir su localización y comunicación.

72. Java

Lenguaje de programación multiplataforma ampliamente utilizado en aplicaciones web, móviles y empresariales.

73. JavaScript

Lenguaje de programación interpretado, esencial en el desarrollo web para crear sitios interactivos y dinámicos.

74. JSON (JavaScript Object Notation)

Formato ligero de intercambio de datos, fácil de leer y escribir para humanos y sencillo de interpretar para las máquinas.

75. LAN (Red de Área Local)

Red que conecta computadoras y dispositivos dentro de un área geográfica limitada, como una oficina o edificio.

76. Arquitectura cliente-servidor

Modelo de red donde el cliente solicita servicios y el servidor los proporciona, común en aplicaciones web y bases de datos.

77. Auditoría Informática

Evaluación sistemática de sistemas tecnológicos para verificar su seguridad y eficiencia.

78. Visión por computadora

Rama de la IA que permite a las máquinas “ver” y analizar imágenes o videos. Se utiliza en sistemas de seguridad, reconocimiento facial, detección de objetos, medicina y autos autónomos.

79. SQL (Structured Query Language)

Lenguaje de programación utilizado para gestionar y consultar bases de datos relacionales de manera eficiente.

80. Byte

Conjunto de 8 bits que representa un carácter, número o símbolo dentro de un sistema digital.

81. Administración de sistemas

Conjunto de prácticas que permiten instalar, configurar, supervisar y mantener servidores y redes para que funcionen correctamente.

82. Análisis de datos

Proceso de inspeccionar, limpiar y modelar información para descubrir patrones, tendencias y tomar decisiones informadas.

83. Metadatos

Información que describe otros datos. Por ejemplo, en una foto los metadatos pueden incluir la ubicación, fecha, cámara utilizada y resolución.

84. Aplicación móvil

Programa diseñado para ejecutarse en smartphones o tabletas, optimizado para pantallas táctiles y sistemas operativos móviles.

85. API REST

Interfaz de programación que permite la comunicación entre sistemas mediante protocolos HTTP, facilitando intercambio de datos en la web.

86. Metaverso

Mundo virtual tridimensional interconectado donde las personas interactúan mediante avatares digitales.

87. Realidad virtual

Tecnología que simula entornos digitales inmersivos mediante visores y sensores de movimiento.

88. Realidad aumentada

Superposición de elementos digitales sobre el entorno físico, integrando lo real y lo virtual.

89. Robótica

Ciencia que diseña, construye y programa robots capaces de realizar tareas de forma autónoma o asistida.

90. Dron

Vehículo aéreo no tripulado controlado a distancia, utilizado para fotografía, logística y vigilancia.

91. Sensor

Dispositivo electrónico que detecta cambios físicos o químicos y los convierte en señales eléctricas.

92. Automatización

Uso de sistemas tecnológicos que sustituyen o reducen la intervención humana en procesos repetitivos.

93. Computación cuántica

Área de la informática que emplea qubits para realizar cálculos extremadamente complejos a gran velocidad.

94. Chip

Circuito integrado diminuto que contiene millones de transistores encargados de procesar información.

95. Nanotecnología

Ciencia que manipula la materia a escala molecular y atómica para crear materiales y dispositivos avanzados.

96. Impresión 3D

Técnica de fabricación aditiva que crea objetos capa por capa a partir de modelos digitales tridimensionales.

97. Red 5G

Tecnología de comunicación móvil de quinta generación que ofrece mayor velocidad, cobertura y baja latencia.

98. ISP (Proveedor de Servicios de Internet)

Empresa que ofrece acceso y conectividad a Internet a usuarios y organizaciones.

99. Ethereum

Plataforma blockchain descentralizada que permite ejecutar contratos inteligentes y aplicaciones distribuidas.

100. Token NFT

Activo digital único basado en tecnología blockchain, usado para certificar propiedad de archivos digitales.

101. Tarjeta gráfica (GPU)

Procesador especializado en renderizar gráficos e imágenes de alta calidad.

102. Disco duro (HDD)

Dispositivo de almacenamiento magnético que guarda datos de forma permanente.

103. Aprendizaje automático (Machine Learning)

Técnica de IA que permite a los sistemas aprender patrones a partir de datos sin ser programados explícitamente.

104. Git

Sistema de control de versiones que permite registrar cambios en el código fuente y colaborar en proyectos de software.

105. Canva

Plataforma de diseño gráfico en línea que permite crear contenido visual sin conocimientos avanzados.

106. HTML5

Última versión del lenguaje HTML, con mejoras para multimedia, gráficos y compatibilidad con dispositivos modernos.

107. Bus de Datos

Canal que transporta información entre los componentes del hardware de un sistema.

108. BIOS (Basic Input Output System)

Sistema que inicia y prueba el hardware del computador antes de cargar el sistema operativo.

109.Modelos preentrenados:

Modelos de IA ya entrenados que pueden adaptarse a nuevas aplicaciones.

110. TIC

Conjunto de tecnologías que permiten la gestión, comunicación y transmisión de información digital.

111. Licencia de software

Permiso legal que regula el uso, distribución y modificación de un programa informático.

112. Log (Bitácora)

Registro de eventos del sistema usado para monitorear actividades y detectar errores.

113. Memory RAM

Memoria de acceso rápido usada por el procesador para almacenar temporalmente datos e instrucciones.

114. Microprocesador

Circuito integrado que realiza las operaciones de cálculo y control dentro de la CPU.

115. Módem

Dispositivo que convierte señales digitales en analógicas y viceversa para conectar a Internet.

116. Nodo

Punto de conexión dentro de una red que puede enviar, recibir o reenviar información.

117. Telefonía fija y móvil

La telefonía fija, también conocida como línea fija o teléfono fijo, es un servicio de comunicación que utiliza una conexión por cable (generalmente de cobre o fibra óptica) para transmitir voz y datos entre dos puntos específicos.

118. Packet (Paquete de datos)

Unidad básica de información que se transmite a través de redes digitales. Contiene datos y dirección de origen y destino para asegurar que llegue correctamente.

119. Password (Contraseña)

Conjunto de caracteres que sirve para autenticar al usuario y proteger la información de accesos no autorizados.

120. Plataforma digital

Entorno virtual que permite ejecutar aplicaciones, servicios o contenidos en línea, como Moodle, Google Classroom o Zoom.

121. Proxy

Servidor intermediario que filtra, controla o almacena el tráfico de información entre usuarios y redes externas para mejorar seguridad o velocidad.

122. RAM (Memoria de acceso aleatorio)

Memoria temporal que permite al computador ejecutar programas y almacenar datos mientras se usan, desapareciendo al apagar el equipo.

123. Repositorio

Lugar centralizado donde se almacena, organiza y gestiona código fuente o archivos de proyectos de software, como GitHub o GitLab.

124. SAAS (Software como servicio)

Modelo de distribución de software que permite usar aplicaciones a través de internet sin instalarlas localmente. Ejemplos: Google Drive, Canva.

125. Seguridad en la nube

Conjunto de prácticas y herramientas que protegen los datos y servicios almacenados en la nube frente a ataques, pérdida o accesos indebidos.

126. Sistema de archivos

Forma en que un sistema operativo organiza y gestiona la información almacenada en discos o memorias, permitiendo acceder a carpetas y archivos.

127. Sistema operativo (SO)

Software que administra los recursos del hardware y proporciona una interfaz entre el usuario y el computador. Ejemplos: Windows, Linux, Android.

128. Software educativo

Aplicaciones y plataformas diseñadas para apoyar el aprendizaje, como Duolingo, Khan Academy o Scratch.

129. Spreadsheet (Hoja de cálculo)

Programa que permite organizar datos en filas y columnas, realizar cálculos y analizar información. Ejemplos: Excel, Google Sheets.

130. SQL

Lenguaje usado para gestionar y consultar bases de datos relacionales. Permite insertar, actualizar, eliminar y buscar información de manera eficiente.

131. SSL/TLS

Protocolos que aseguran la transmisión de datos en internet mediante cifrado, protegiendo información como contraseñas o datos bancarios.

132. Autenticación Multifactor (MFA)

Método de seguridad que requiere más de una forma de verificación.

133. Streaming en vivo

Transmisión de audio o video en tiempo real, permitiendo a los usuarios visualizar contenido mientras se produce. Ejemplos: YouTube Live, Twitch.

134. Switch de red

Dispositivo que conecta múltiples dispositivos dentro de una red local y permite enviar información de manera eficiente entre ellos.

135. Sistema embebido

Computador especializado integrado dentro de un dispositivo para realizar tareas concretas, como control de electrodomésticos o automóviles.

136. Terminal de comandos

Interfaz de texto que permite interactuar con el sistema operativo o aplicaciones mediante comandos, útil para administración y programación avanzada.

137. Usuario final

Persona que utiliza directamente aplicaciones o sistemas sin necesidad de conocer su funcionamiento interno.

138. Virtual Machine (Máquina virtual)

Sistema simulado por software que permite ejecutar un sistema operativo completo dentro de otro, útil para pruebas y desarrollo seguro.

139. VPN (Red privada virtual)

Tecnología que crea una conexión segura y cifrada entre dispositivos y redes a través de internet, protegiendo datos y privacidad.

140. Web server (Servidor web)

Computador o software que entrega contenido web a los navegadores de los usuarios, permitiendo el funcionamiento de páginas y aplicaciones en línea.

141. Wi-Fi

Tecnología de conexión inalámbrica que permite a los dispositivos acceder a internet y redes locales sin cables.

142. XML (eXtensible Markup Language)

Lenguaje de marcado que permite almacenar y transportar información estructurada, facilitando el intercambio de datos entre sistemas distintos.

143. XPath

Lenguaje usado para buscar y seleccionar información específica dentro de documentos XML, común en programación y desarrollo web.

144. YAML

Lenguaje de serialización de datos legible para humanos, usado en configuraciones de software y aplicaciones de desarrollo.

145. Zoom

Plataforma de videoconferencias que permite reuniones virtuales, clases en línea y colaboración a distancia.

146. Conexión física

Requiere una conexión por cable para enlazar el teléfono con la central telefónica.

147. Ubicación fija

El servicio está atado a una dirección específica, lo que significa que no ofrece movilidad al usuario.

148. Redes

Una red es un sistema de interconexión que permite la comunicación y el intercambio de datos entre dos o más dispositivos electrónicos, mediante medios físicos o inalámbricos, compartiendo recursos y facilitando la comunicación sin importar la distancia entre los nodos.

149. Terminales

Son los dispositivos o equipos que utilizamos para acceder a la información y a los servicios que las TIC ofrecen.

150. TIC en el ámbito empresarial

Tecnologías que mejoran la gestión y la productividad de una empresa.

151. Servicios

Aplicaciones y herramientas que los usuarios utilizan a través de los terminales y las redes para realizar diversas tareas.

152. Código abierto (Open Source)

Software cuyo código es accesible públicamente, permitiendo usarlo, modificarlo y redistribuirlo libremente.

153. IDE (Entorno de Desarrollo Integrado)

Aplicación que facilita la programación, combinando editor de código, compilador y depurador en un solo entorno. Ejemplos: PyCharm, Visual Studio Code.

154. Sesgo algorítmico

Ocurre cuando los algoritmos de inteligencia artificial y aprendizaje automático producen resultados injustos o discriminatorios debido a datos de entrenamiento sesgados o diseños erróneos.

155. Gateway

Dispositivo que conecta dos redes diferentes, permitiendo la comunicación y traducción de protocolos entre ellas.

156. Linux

Sistema operativo de código abierto basado en Unix, usado en servidores, computadoras personales y dispositivos embebidos.

157. Microcontrolador

Dispositivo electrónico con CPU, memoria y periféricos integrados, usado para controlar sistemas electrónicos específicos.

158. RAM (Random Access Memory)

Memoria temporal que permite al computador ejecutar programas y acceder a datos rápidamente mientras están en uso.

159. Control de versiones

Sistema que registra cambios realizados en archivos o proyectos de software, permitiendo gestionar versiones y colaborar en equipo. Ejemplo: Git.

160. CRM (Customer Relationship Management)

Software que ayuda a gestionar relaciones con clientes, registrar interacciones y analizar información para mejorar ventas y servicios.

161. Data Mining (Minería de datos)

Proceso de analizar grandes cantidades de datos para extraer información útil, descubrir patrones y tomar decisiones estratégicas.

162. Depuración (Debugging)

Actividad de identificar y corregir errores en un programa o aplicación para asegurar que funcione correctamente.

163. Desarrollo ágil

Metodología de gestión de proyectos de software basada en iteraciones cortas, colaboración constante y entrega frecuente de funcionalidades.

164. DNS (Sistema de nombres de dominio)

Sistema que traduce nombres de dominio legibles por humanos en direcciones IP que las computadoras pueden entender.

165. Email (Correo electrónico)

Sistema de envío y recepción de mensajes digitales a través de internet, con capacidad de adjuntar archivos y comunicarse instantáneamente.

166. Edge AI

Rama de la inteligencia artificial que consiste en ejecutar algoritmos y modelos de aprendizaje automático directamente en dispositivos cercanos al usuario, como teléfonos inteligentes, cámaras de seguridad, sensores industriales o vehículos autónomos, en lugar de depender completamente de servidores en la nube. Esto permite procesar datos en tiempo real, mejorar la velocidad de respuesta, reducir la latencia y disminuir la necesidad de enviar grandes volúmenes de información a centros de datos remotos. Además, Edge AI incrementa la privacidad y la seguridad de los datos al mantener la información sensible localmente, y facilita la operación en entornos con conectividad limitada o intermitente.

167. Encriptación

Se denomina centro de procesamiento de datos al edificio o sala de gran tamaño usada para mantener en él una gran cantidad de equipamiento informático y electrónico.

1318. Syslog

Es un protocolo estándar para el registro de mensajes de eventos y el sistema en entornos de TI. Permite que los dispositivos de red, sistemas operativos (como Linux) y software envíen mensajes a un servidor centralizado, lo que facilita la monitorización, el análisis y la solución de problemas.

1319. ZigBee

Es un protocolo de comunicación inalámbrica de bajo consumo y bajo coste, ideal para el Internet de las Cosas (IoT) y la domótica. Se basa en el estándar IEEE 802.15.4 y se distingue por su arquitectura de red en malla, que aumenta su alcance y fiabilidad. A diferencia de las redes Wi-Fi, Zigbee está optimizado para la transmisión de pequeños paquetes de datos, como los que se envían desde sensores y controles.

1320.CoAp

Es un protocolo de capa de aplicación para la Internet de las cosas (IoT) que permite la comunicación entre dispositivos con recursos limitados, como microcontroladores, utilizando un modelo de solicitud-respuesta similar a HTTP pero basado en UDP para una menor sobrecarga.

1321. Dynamic Routing

El Dynamic routing hace que las actualizaciones automáticas enruten tablas a medida que cambia la configuración de una red.

1324. Red privada

Una red privada es una red de computadoras que utiliza un espacio de direcciones IP privadas, lo que la aísla y protege del acceso directo de la internet pública. Dentro de esta red, los dispositivos pueden comunicarse entre sí de forma segura y controlada.

1325.Telnet

Permite a los usuarios conectarse y controlar ordenadores de forma remota, como si estuvieran sentados físicamente frente a ellos. A pesar de su antigüedad y sus importantes fallos de seguridad, todavía se utiliza en entornos específicos.

1326.ARP (Address Resolution Protocol)

Es un protocolo de red que mapea una dirección IP (capa de red) a su dirección MAC física (capa de enlace) dentro de una red local (LAN) para permitir la comunicación entre dispositivos.

1327. La puerta de enlace predeterminada

La puerta de enlace predeterminada es un dispositivo (generalmente un router) que un ordenador utiliza para enviar información a otras redes o a Internet, actuando como un punto de acceso y enrutamiento de datos cuando no hay una ruta específica definida.

1328. Subred (Subnet)

Una subred es una subdivisión lógica de una red IP más grande, creada mediante el proceso de subredeo (subnetting) para mejorar la gestión del tráfico, la seguridad y la eficiencia en la asignación de direcciones IP.

1329. Repetidor

Un repetidor es un dispositivo electrónico que recibe una señal (como la de Wi-Fi) y la transmite de nuevo, amplificada o regenerada, para extender su alcance a través de distancias mayores o para superar obstáculos

1330. Hub

Un "hub" puede referirse a un dispositivo de red que conecta múltiples equipos, a un centro de distribución de personas o mercancías (como un aeropuerto o puerto), o a un centro o punto focal donde se concentran actividades, talento o servicios, como un hub tecnológico, cultural o de innovación.

1331. Estrella informatica

Una estrella informática se refiere a la topología en estrella, una arquitectura de red en la que todos los dispositivos (como computadoras, impresoras, etc.) se conectan individualmente a un punto central (como un concentrador o conmutador) que gestiona la transmisión de datos

1332. Cloud key management service (KMS)

Servicio que gestiona claves de cifrado de forma segura para proteger información en entornos cloud.

1333. Cable de cobre

Un cable de cobre es un conductor eléctrico compuesto por uno o más hilos de cobre, un metal altamente dúctil y con una excelente conductividad eléctrica. A menudo están protegidos por un revestimiento aislante para evitar cortocircuitos y otros riesgos eléctricos.

1334. Autenticación multifactor.

La autenticación multifactor (MFA) añade una capa extra de seguridad al proceso de inicio de sesión, requiriendo al menos dos formas de verificación para acceder a una cuenta, como una contraseña y un código en el teléfono o una huella dactilar. Esto protege las cuentas incluso si la contraseña es comprometida, haciendo mucho más difícil el acceso no autorizado.

1336. ondas electromagnéticas

Las ondas electromagnéticas son perturbaciones de campos eléctricos y magnéticos que se propagan a la velocidad de la luz, transportando energía a través del espacio o de un medio materia

1339. Teclado

Un teclado es un dispositivo de entrada para ordenadores y otros aparatos electrónicos que permite introducir texto, números y símbolos mediante la pulsación de sus teclas, que actúan como interruptores electrónicos. Generalmente se organiza en bloques: alfanumérico (letras y números), de funciones (F1, F2, etc.), numérico (para números) y especial (como Ctrl, Alt, etc.).

dispositivo de entrada de computadora que te permite controlar un puntero en la pantalla y ejecutar acciones al interactuar con la superficie. Generalmente, tiene botones para hacer clic y una rueda de desplazamiento para navegar por documentos.

1. Algoritmo

Colección organizada de datos que permite almacenar, consultar y actualizar información de manera eficiente.

5. Bitácora

7. Bug

Modelo tecnológico que permite almacenar, gestionar, procesar y acceder a datos, programas y servicios a través de Internet.

Conjunto de instrucciones escritas en un lenguaje de programación para que un computador realice tareas específicas.

Programa que traduce el código escrito en un lenguaje de alto nivel a un lenguaje máquina ejecutable.

El cerebro del computador que ejecuta instrucciones y procesa datos.

12. Data (Datos)

Conjunto de hechos, cifras o información que pueden ser procesados por sistemas informáticos.

13. Decimal (Número decimal)

14. Debugging

Proceso de identificar, analizar y corregir errores en un programa o aplicación.

Sistema de seguridad informática que controla el tráfico de datos entre una red interna y externa.

Parte central de un sistema operativo que gestiona recursos, memoria, procesos y comunicación entre hardware y software.

Conjunto de dispositivos y protocolos que permiten la comunicación entre computadores.

Software cuyo código es abierto y accesible para que cualquiera pueda usarlo, modificarlo y distribuirlo.

Lenguaje de programación de alto nivel, sencillo y versátil, usado para desarrollo web, análisis de datos e inteligencia artificial.

Computador o programa que proporciona servicios y recursos a otros dispositivos o usuarios en una red.
